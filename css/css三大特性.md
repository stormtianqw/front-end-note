## 继承性

继承性是指指被包在内部的标签将拥有外部标签的样式性，即子元素可以继承父元素的属性。在CSS中以text-、font-、line- 开头的属性都是可以继承的。但是也有以下几个特殊点：
- a 标签的字体颜色不能被继承
-	h标签字体的大下也是不能被继承的

故a、h标签在common.css文件中会被单独列出
