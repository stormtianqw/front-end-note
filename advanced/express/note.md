2 检测代码变动，不用实时在终端执行node `npm install supervisor –g`

3 静态服务器，随时随地将当前目录变成一个静态文件服务器的根目录 `npm install anywhere –g`
[https://github.com/JacksonTian/anywhere](https://github.com/JacksonTian/anywhere)
