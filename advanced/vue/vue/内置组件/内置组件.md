# 内置组件

## component

动态组件，该组件有个 prop 叫 is，注意区分特殊 attribute is，后者是用于解决有些 HTML 元素，诸如 `<ul>`、`<ol>`、`<table>` 和 `<select>`其内部只能出现特定元素的限制的问题。
